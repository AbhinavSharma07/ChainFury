chainfury Components
====================

`chainfury` is designed to work OOB with several popular vendor APIs and integrates to several components in the larger
ecosystem. You can talk to the components directly but we recommend wrapping them in a `chainfury` action or via
`ai_actions_registry` or `programatic_actions_registry`.
