Mathematically, the bias is added to the logits generated by the model prior to sampling. The exact effect will vary per model, but values between -1 and 1 should decrease or increase likelihood of selection; values like -100 or 100 should result in a ban or exclusive selection of the relevant

  • user ‚Äì Optional. A unique identifier representing your end-user, which can help OpenAI to monitor and detect abuse. Defaults to None.
